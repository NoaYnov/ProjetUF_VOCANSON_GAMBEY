Bienvenue sur le projet UF de Vocanson Matteo et de Gambey Noa

Le but de ce projet était donc de créer un total de 5 VM (Virtual Machine):
- Une Vm Server Web (Apache2), sans interface graphique
- Une Vm Server de Backup (Rsync), sans interface graphique
- Une Vm Client Linux (Debian), avec interface graphique
- Une Vm Client Windows (Windows 10), avec interface graphique
- Une Vm Parefeu (Pfsense), sans interface graphique
